# WebAuthnDemo

An example TypeScript Relying Party implementation of the [WebAuthn
specification](https://w3c.github.io/webauthn/).

## Install

Checkout the repository, then install.

```sh
$ npm install
```

## Build

Bulid the project.

```sh
$ npm run build
```

## Start a local server

Run the Firestore emulator:

```sh
$ npm run emulator
```

Run the server:

```sh
$ npm run dev
```
